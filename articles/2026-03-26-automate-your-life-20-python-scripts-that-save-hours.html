2. Data Analysis with Pandas

Data analysis is crucial for making informed decisions. Python's Pandas library makes it easy to manipulate and analyze data.

        import pandas as pd

        # Load the data
        df = pd.read_csv('data.csv')

        # Group by a column and calculate the mean
        grouped = df.groupby('column').mean()

        # Save the results
        grouped.to_csv('grouped_data.csv')

Here, we load a CSV file, group the data by a specific column, calculate the mean, and save the results. This can be useful for summarizing large datasets.

3. Website Scraping with BeautifulSoup

Scraping data from websites can be a time-consuming task. BeautifulSoup is a powerful library for parsing HTML and XML documents.

        from bs4 import BeautifulSoup
        import requests

        # Fetch the webpage
        response = requests.get('https://example.com')
        soup = BeautifulSoup(response.text, 'html.parser')

        # Find all the links
        links = soup.find_all('a')

        # Print the links
        for link in links:
            print(link.get('href'))

This script fetches a webpage, parses it, and extracts all the links. This can be useful for web scraping projects or monitoring the content of a website.

4. Backup Your Files

Regular backups are essential to prevent data loss. Python can help you automate the process of backing up your files.

        import os
        import shutil

        # Define the source and destination paths
        source = '/path/to/source'
        destination = '/path/to/destination'

        # Loop through all the files in the source directory
        for filename in os.listdir(source):
            # Create the full source and destination paths
            source_path = os.path.join(source, filename)
            destination_path = os.path.join(destination, filename)

            # Copy the file to the destination
            shutil.copy(source_path, destination_path)

This script copies all files from the source directory to the destination directory. You can schedule this script to run regularly using cron jobs or Task Scheduler.
